# Growth-Analysis(Interactive Dashboard creation using Microsoft Excel and Power BI)

## Project Objective
  This project analyzes financial performance across regions, products, and months using Excel PivotTables, charts, KPI cards, and slicers.The dashboard provides Total Revenue,Total Profit,Total Expenses,Profit Margin%,Units Sold,Region-wise Insights,Month-wise trends.
## Dataset used:
- <a href="https://github.com/Varshanatchathra/Financial-growth-Analysis/blob/main/FinanceData.xlsx" >Dataset</a>

## Tools Used:
- Microsoft Excel (PivotTables, Pivot Charts, Slicers, KPI Cards)
- Power BI (DAX,Slicers, charts for Visualization)
- Basic Finance Metrics (Revenue, Profit, Expenses, Profit Margin)

## Insights:

- From the dashboard:
- Region with Highest Revenue - East region has the Highest revenue.
- Most Lucrative Product - Loan generates the highest revenue.
- Revenue Trend - Slight rise in early months,then fluctates.
- Highest Profit Month - Augest & November show profit peaks.
- Most Stable Region - East shows consistent performance.
- Lowest Expenses - Credit Card category has the lowest expenses.

  
- Dashboard Interaction <a href="https://github.com/Varshanatchathra/Financial-growth-Analysis/blob/main/FinanceData%20-%20Excel%2030-03-2026%2009_20_28%20PM.png">View Dashboard</a>

## Dashboard Features:
- KPI cards - Total Revenue,Total Profit,Total Expenses,Unit Sold,Profit Margin%
- Charts Include - Revenue by Region,Profit Month-wise,Revenue Month-wise,Expenses by Product,Region-wise Monthly Trends,Product-wise Revenue
- Slicer Filters - Region,Product,Month

## Process

- Loaded financial dataset(Revenue,Profit,Expenses,Units Sold)
- Cleaned the dataset(blank,duplicates)
- Created Pivot Tables for Region Revenue,Product Revenue,Monthly Profit.
- Designed Pivot charts for visual interpretation.
- Added slicers for interactive filtering.
- Created KPI cards using shapes & Formulas.
- Applied color theme & layout styling.

## Final Conclusion:
- From the analysis ,Revenue stablility should be improved While several months show strong revenue, the dashboard reveals fluctuations mid-year.
Using strategies like seasonal marketing, targeted promotions, and customer retention programs can reduce volatility.
- Focus on Growth-Inconsistent Regions-Regions like North and South display moderate and inconsistent performance.
Targeted improvement strategies, such as localized campaigns or product customization, can help stabilize growth.
- Operational expenses should be reduced , especially in cost-heavy categories.
